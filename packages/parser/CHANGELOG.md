# Changelog

## 0.1.0

### Minor Changes

- d343a2d: Initial public 0.1.0 release. The parser, registry, and spec packages are now publishable as the metadata foundation of `dtcg-formulas`. Adds dual ESM/CJS builds via tsup, full `exports` map, npm publish metadata, and the public-readiness architecture analysis. Resolver, CLI, and executable adapters land in 0.2.0; see the [roadmap](https://dtcg-formulas.org/guide/roadmap) for the full plan.

All notable changes to `@dtcg-formulas/parser` will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).
