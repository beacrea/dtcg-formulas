# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

By participating in this project (filing issues, opening pull requests, posting in discussions), contributors agree to uphold its standards.

## Reporting

Concerns about conduct in this project's spaces (issues, PRs, discussions) can be reported by opening an issue tagged `conduct`, or by emailing the maintainer listed in [`SECURITY.md`](./SECURITY.md). Reports will be handled with discretion.
